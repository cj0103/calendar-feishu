# 变更日志

所有重要的项目变更都将记录在此文件中。

## [1.3.0] - 2026-03-17

### ✨ 新增功能
- **隐藏终端启动**：提供 VBScript 脚本，双击即可启动开发服务器，终端不显示在任务栏
  - `start-dev.vbs` - 隐藏终端启动（无日志）
  - `start-dev-logging.vbs` - 隐藏终端启动（带日志记录）
  - `start-dev.ps1` - PowerShell 隐藏启动
  - `创建桌面快捷方式.bat` - 一键创建桌面快捷方式
- **VS Code 任务配置**：集成后台任务启动，支持快捷键启动
- **飞书全量同步按钮**：在飞书日历管理中心添加"📥 从飞书同步"按钮
  - 支持从飞书全量下载日历日程
  - 智能更新已存在的日程
  - 自动清除对应日历的同步队列

### 🔄 优化改进
- **简化同步逻辑**：重构 SyncManager，避免重复同步问题
  - 启动时只添加本地独有事件（无 feishuEventId）到队列
  - 移除 incrementalSync 方法，不再重复检测本地独有事件
  - sync() 方法简化，只返回队列统计
- **同步队列优化**：优化队列管理，只在本地操作时添加到队列
  - 新增日程 → 自动加入 create 队列
  - 编辑日程 → 自动加入 update 队列
  - 删除日程 → 自动加入 delete 队列
- **文档整合**：整合启动说明文档，提供完整的使用指南

### 📝 文档更新
- 更新 README.md，添加隐藏终端启动说明
- 新增 QUICK_START.md - 快速启动指南
- 新增 启动说明.md - 详细启动说明
- 更新 .trae/documents/简化同步逻辑方案.md

### 🐛 Bug 修复
- 修复同步按钮显示重复计数的问题
- 修复启动时队列重复添加已同步事件的问题
- 修复飞书同步后队列未清除的问题

### 🔧 技术改进
- 移除无用的调试代码
- 优化代码结构和注释
- 改进错误处理和日志输出
